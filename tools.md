# List of downstream tools

A key point we want to make is that tskit is now essential infrastructure for
the field, and so we want to assemble a list of such tools. The criteria for
inclusion is

Incorporates tskit/msprime APIs somewhere in the production codebase (not just
for testing)
Supports tskit tree sequence files as an input/output format.
Is published at least as a preprint
We'll split the list up into four categories: simulation, inference,
visualisation and analysis.
